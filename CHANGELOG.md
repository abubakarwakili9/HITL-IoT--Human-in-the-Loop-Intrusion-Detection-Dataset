# Changelog

All notable changes to the HITL-IoT dataset will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2025-12-09

### Added
- Initial release of HITL-IoT dataset
- 127,845 labeled IoT network flows
- 10,227 human-annotated decisions
- 12 device types and 12 attack categories
- 9 ML baseline implementations
- Complete documentation and examples
- Citation file (CITATION.cff)
- CC BY 4.0 license

### Features
- Expert, intermediate, and novice annotations
- Confidence scores and decision times
- ML model predictions and confidence
- 54 CICFlowMeter-style features
- Reproducible generation (seed=42)
